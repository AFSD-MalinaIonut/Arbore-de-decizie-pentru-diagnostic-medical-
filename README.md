#  Diagnostic Medical Simplificat - Arbore de Decizie

##  Descrierea Proiectului
Acest proiect reprezintă o aplicație practică a algoritmilor de Machine Learning (Arbori de Decizie) în domeniul medical. Scopul principal este de a dezvolta un model capabil să ofere un diagnostic prezumtiv simplificat pentru afecțiuni respiratorii comune (Răceală, Gripă, COVID-19, Alergii), bazându-se pe un set de simptome introduse de utilizator.

Proiect realizat în cadrul disciplinei **Practica de specialitate**.
